## вңЁ How It Works

1. **Create** вҖ” Enter name, birthday message, pick a theme
2. **Generate** вҖ” Get a QR code with the surprise embedded
3. **Share** вҖ” Print the QR, send it, or put it on a gift
4. **Reveal** вҖ” When scanned, reveals a full interactive birthday experience!

## рҹҺҜ Features

- **рҹҺЁ 8 Beautiful Themes** вҖ” Sparkle, Galaxy, Sakura, Ocean, Sunset, Forest, Candy, Royal
- **рҹ“ұ QR Code Generator** вҖ” Download QR image, share link

- **рҹҺө Birthday Melody** вҖ” Web Audio API generated music
- **рҹ’қ Wish Cards** вҖ” Tap-to-reveal surprise wishes

- **вңЁ Floating Particles** вҖ” Theme-matched ambient particles
- **рҹ“қ Personal Messages** вҖ” Custom name, age, message, sender
- **рҹ”— URL Hash Encoding** вҖ” All data encoded in URL (no server needed)
- **рҹ“І Mobile First** вҖ” Optimized for phone screens
